Introduction of i

      

          The imaginary number i is set at a value of the square root of -1. This number is used in a variety of equations, most used when solving for zeroes.
          Allow us to describe some examples

          sqrt(-7) = 7i
          sqrt(-12) = 12i
          3 * sqrt(-2) = 6i

          i follows the following pattern when it comes to powers of i:
          i^1 = i
          i^2 = -1
          i^3 = -i
          i^4 = 1

          Examples:
          i^7 = -1
          i^15 = -1
          i^2024 = 1
