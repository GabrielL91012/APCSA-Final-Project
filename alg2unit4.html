Properties of exponents (rational exponents)

      

          Properties of exponents contains needing to know how to properly factor. Using formulas and examples from the previous units, we can easily determine answers for these:

          Example:
          (a^(2/3))^-1 = a^(-2/3)
        
     

     
Solving Exponential Equations using properties of exponents

    

          We can solve these equations using logarithms using this formula

          log base a(x) = b          a^b = x

          Example:
          27^(3x-7) = 9^(6x-1)
          9^(2/3) = 27
          (2/3)(3x-7) = 6x-1
          6x-14 = 18x-3
          -12x=11
          x=-11/12
